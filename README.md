#### Projects

- **[Pi Server](https://github.com/marcfargas/pi-server)** — Detachable agent sessions for pi — headless daemon + terminal client over WebSocket.
- **[Pi Subagents](https://github.com/marcfargas/pi-subagents)** — Pi extension for async subagent delegation with truncation, artifacts, and session sharing.
- **[Skills](https://github.com/marcfargas/skills)** — Reusable skills for AI coding agents (pi, Claude Code, Agent Skills standard).
- **[Pi Extensions](https://github.com/marcfargas/pi-mf-extensions)** — Persistent, auditable plan-then-execute workflow for pi agents.
- **[Odoo Toolbox](https://github.com/marcfargas/odoo-toolbox)** — CLI toolkit for Odoo ERP. Built for AI agent integration.
- **[Go Easy](https://github.com/marcfargas/go-easy)** — Unified CLI for Gmail, Drive, Calendar, and Tasks. Google APIs without the pain.
- **[Go MAPI](https://github.com/marcfargas/go-mapi)** — Right-click any file in Windows Explorer, "Send to Mail recipient", and the email opens in Gmail. Zero config, open source.
